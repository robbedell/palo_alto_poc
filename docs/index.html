Custom Interface Configuration

Add custom interfaces by selecting the interface name, and optionally specifying the IP address and subnet mask. Default configuration is cleared when custom values are added.

Custom Address Objects

Add custom address objects by specifying the name, IP address, and description. Default configuration is cleared when custom values are added.

Custom Service Objects

Add custom service objects by specifying the name, protocol, port, and description. Default configuration is cleared when custom values are added.
